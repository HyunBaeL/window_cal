# 📌 계산기 프로젝트 기능 명세서

## 👥 역할 분담

| 이름     | 담당 역할                                     |
|----------|-----------------------------------------------|
| **공동개발** | UI 구현 |
| **홍아랑** | 고급 연산(루트, 로그, 제곱, 파이 등) 기능 개발 |
| **이현배** | 기본 연산(사칙연산, 부호, 클리어), 입력 처리 및 이벤트 로직 구현 |

---

## 🧮 핵심 기능 정의

### 🔢 기본 연산
- ➕ 더하기  [Add]
- ➖ 빼기  [Sub]
- ✖️ 곱하기  [Mul]
- ➗ 나누기 + 나머지 (a ÷ b = q … r)  [Div]

### 🧠 고급 수학 연산
- √ 루트 (제곱근)  [Square]
- xʸ 제곱 (x^y)  [PowCal]
- 🔟 10의 제곱 (10 × x)  [ExpCal]
- log 로그 계산 (log₁₀)  [LogCal]
- π 파이 상수 입력  [PiCal]

### 🧾 수 관련 변환
- ➗ 분수 변환 (ex: 0.5 → 1/2)  [FracCal]
- 🔁 절대값 (`Math.Abs(x)`)  [AbsCal]
- 🔃 부호 변환 (x → -x)  [SignCal]

### ⏳ 기타 연산
- ❗ 팩토리얼 (n!)  [PactCal]
- 🧼 클리어 (모든 값 초기화)  [Clear]
- ⬅️ 한 자리씩 지우기 (Backspace)  [Del]
- 🔘 소수점 입력  [DemPoint]
- ( 왼쪽 괄호  [LeftB]
- ) 오른쪽 괄호  [RightB]
- `=`: 계산 결과 출력
